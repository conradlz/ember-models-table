Class PageSizeSelect

Dropdown with page size values used within TableFooter.

Value from ModelsTable.pageSizeOptions is used as a list of dropdown-options.

Usage example:

<ModelsTable @data={{this.data}} @columns={{this.columns}} as |MT|>
  <MT.Footer as |Footer|>
    <Footer.PageSizeSelect />
    {{! ... }}
  </MT.Footer>
  {{! .... }}
</ModelsTable>

Block usage example:

<ModelsTable @data={{this.data}} @columns={{this.columns}} as |MT|>
  <MT.Footer as |Footer|>
    <Footer.PageSizeSelect as |SizeSelectBlock|>
